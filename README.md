# 🛍️ Florza - E-Commerce Website

## 📖 Project Overview

Florza is a modern and user-friendly E-Commerce Website developed using HTML, CSS, and JavaScript.

The website allows users to browse products, search for products, filter items by category, add products to a shopping cart, remove products from the cart, and place orders seamlessly.

This project was developed as part of my Full Stack Development Internship to improve my front-end development skills and gain hands-on experience with JavaScript DOM Manipulation.

## 🚀 Functionalities Implemented

### 🛍️ Product Display

All products are dynamically displayed using JavaScript.

### 🔍 Search Products

Users can search products instantly using the search bar.

### 📑 Category Filtering

Products can be filtered based on categories:

👩 Women

👨 Men

🧒 Kids

🐶 Pets

💻 Gadgets

🛒 Grocery

### 🛒 Shopping Cart

Users can:

➕ Add products to cart

❌ Remove products from cart

📊 View cart item count

💰 View total price

### 📦 Order Placement

Users can place orders successfully after adding products to the cart.

### 💾 Local Storage

Cart data is stored using browser local storage so that products remain available even after page refresh.
